# ml-pbl
# Car Rating Predictor 🚗

A Machine Learning web application that predicts a car's rating based on specific features. This project demonstrates a full-stack integration of an AIML model with a Flask backend and a modern web frontend.

---

## 🚀 Project Overview
This project was developed as part of **Project-Based Learning (PBL)** to bridge the gap between data science models and user-accessible interfaces. It uses a regression/classification model to analyze car attributes and provide an estimated rating.

### Key Features:
* **Machine Learning Backend:** Built with Python and Flask.
* **Responsive Frontend:** Clean UI using HTML5, CSS3, and JavaScript.
* **Real-time Prediction:** Communicates via REST API to fetch model results.

🛠️ Installation & Setup
1. Clone the Repository
Bash
git clone [https://github.com/pavanyadavg99-lab/ml-pbl.git](https://github.com/pavanyadavg99-lab/ml-pbl.git)
cd ml-pbl
2. Setup Backend
Navigate to the backend folder and install the required Python libraries:

Bash
cd backend
pip install -r requirements.txt
3. Run the Application
Start the Flask Server:

Bash
python app.py
The backend will typically run on http://127.0.0.1:5000

Open the Frontend:
Navigate to the frontend/ folder and open index.html in your preferred web browser.
